Kullanım Koşulları

Lütfen emirhanuzgun.dev web sitesini kullanmadan önce aşağıdaki koşulları okuyunuz.

1. Telif Hakkı

Bu web sitesinde yer alan tüm tasarımlar, kod yapıları, proje görselleri ve içerikler (aksi belirtilmedikçe) Emirhan Üzgün'e aittir. Kişisel portfolyo amacı taşımaktadır. İzinsiz kopyalanamaz veya ticari amaçla çoğaltılamaz.

2. İçerik Kullanımı

Sitede sergilenen açık kaynak (open-source) projeler, ilgili GitHub depolarında belirtilen lisans koşullarına (MIT, Apache vb.) tabidir. Diğer tüm içerikler sadece inceleme ve referans amaçlıdır.

3. Sorumluluk Reddi

Web sitesinde yer alan bilgiler genel bilgilendirme amaçlıdır. Bu site üzerinden verilen dış bağlantıların (örneğin LinkedIn, Instagram vb.) içeriğinden veya güvenliğinden site sahibi sorumlu tutulamaz.

4. Değişiklikler

Site sahibi, bu koşulları dilediği zaman önceden haber vermeksizin değiştirme hakkını saklı tutar.
